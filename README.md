# Controle de Horas Trabalhadas

Projeto Implementado durante a seleção de estágio para a BlockHub Builders

### Instalação e execução do projeto usando 'yarn'

```
yarn install
yarn serve
```

### Instalação e execução do projeto usando 'npm'

```
npm install
npm run serve
```
